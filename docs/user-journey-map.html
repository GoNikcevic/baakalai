Bakal — Parcours Utilisateur

User Journey Map · État actuel de la plateforme · 7 étapes du premier contact à la boucle d'optimisation

1
🚪 Onboarding — Wizard 3 étapes Première connexion

L'utilisateur arrive pour la première fois. Un wizard plein écran le guide en 3 étapes : upload de documents business (plaquettes, études de cas), connexion des APIs (Lemlist, Claude, Notion), et configuration initiale du profil. Le dashboard affiche ensuite un bandeau de bienvenue avec checklist.

📄 Drag & drop documents 🔑 Saisie clés API ✅ Checklist 4 étapes
Émotion
Curieux mais hésitant
2
🏢 Profil Entreprise Page Profil

Le client renseigne son identité business : proposition de valeur, preuves sociales, pain points de ses cibles, personas (décideur, secteur, taille, zone géo), et préférences de copy (ton, formalité, vocabulaire). Ces infos alimentent directement la génération IA des séquences.

✏️ Valeur prop & preuve sociale 🎯 Personas cibles 🗣️ Ton & formalité 📎 Upload documents
Émotion
Concentré, investi
3
🚀 Création de campagne Modal création

Via le bouton "Nouvelle campagne", une modal s'ouvre. Le client choisit : secteur cible, poste visé, zone géo, canal (Email / LinkedIn / les deux), angle d'approche (pain point, preuve sociale, éducatif…), volume de prospects. Option "Inspiration" qui redirige vers l'assistant IA pour brainstormer.

📝 Formulaire campagne 🎯 Choix angle & canal 💬 Assistant IA (optionnel) 📊 Volume prospects
Émotion
Motivé, décisionnel
4
✍️ Édition Copy & Séquences Page Copy Editor

Éditeur split-panel : liste des campagnes à gauche, touchpoints à droite. Chaque message (E1→E4, LinkedIn) est éditable inline avec variables {{firstName}}, {{companyName}}. Des suggestions IA apparaissent en contexte avec rationale — le client peut Accepter, Modifier ou Ignorer. Mode preview avec données fictives.

📧 Édition inline 🤖 Suggestions IA 👁️ Mode preview 🔤 Variables dynamiques 📏 Compteur 300 chars LinkedIn
Émotion
Créatif, en contrôle
5
📊 Suivi quotidien — Dashboard Page Dashboard

Vue d'ensemble avec 6 KPIs (prospects contactés, ouverture, réponse, intéressés, RDV, valeur estimée), tableau des campagnes actives, graphe de performance 4 semaines, opportunités récentes, et widget "Recommandations Claude". Notifications en temps réel (alertes perf, RDV, analyses).

📈 6 KPIs temps réel 📋 Tableau campagnes 📉 Graphe tendance 🔔 Notifications 💡 Recommandations
Émotion
Confiant, informé
6
🧪 Optimisation — Refinement A/B Page Refinement

Comparaison visuelle Variant A (contrôle) vs Variant B (challenger) avec métriques côte à côte. Stats globales : variants testées, gain moyen, golden nuggets identifiées. Timeline d'apprentissage : patterns validés, angles abandonnés, insights cross-campagnes avec niveaux de confiance.

⚔️ A/B comparaison 💎 Golden nuggets 🧠 Mémoire apprentissage 📊 Niveaux de confiance
Émotion
Excité par les résultats
7
🔄 Boucle continue — Auto-amélioration Toutes les pages

Le système boucle automatiquement : stats collectées → Claude analyse → recommandations inline dans Copy → nouveau test A/B → apprentissages consolidés dans la mémoire cross-campagne. Chaque nouvelle campagne bénéficie de tout ce qui a été appris. Le client supervise, le système optimise.

⚡ Stats auto (Lemlist → N8N) 🤖 Analyse Claude ✍️ Régénération copy 🧠 Mémoire consolidée
Émotion
Autonome, confiant
